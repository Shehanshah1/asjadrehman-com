## üõ† Running Locally

This app requires **Node.js v18.17+** and optionally uses `bun` for dev speed.

```bash
git clone https://github.com/Shehanshah1/asjadrehman-site.git
cd asjadrehman-site

# with bun (preferred)
bun install
bun dev

# or with npm
npm install
npm run dev
```
